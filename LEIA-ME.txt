HERO WEBGL - Projeto isolado na Área de Trabalho

Nome da pasta: Hero- webgl  (com espaço entre "Hero-" e "webgl")

Onde procurar na Área de Trabalho:
• Se sua Área de Trabalho é a do OneDrive:  C:\Users\felli\OneDrive\Desktop\Hero- webgl
• Se é a Área de Trabalho local:            C:\Users\felli\Desktop\Hero- webgl

A pasta fica FORA de qualquer outra (ao lado de "aegis", não dentro dela).

Para rodar:
1. Abra esta pasta no terminal
2. npm install
3. npm run dev
4. Abra http://localhost:3000 no navegador

Contém: Hero com WebGL, partículas, ondas, fundo #0A0B0D.
